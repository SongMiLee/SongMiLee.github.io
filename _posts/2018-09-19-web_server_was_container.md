### Web Server
- 클라이언트의 요청을 받아 정적인 페이지(html, css, image)를 HTTP 프로토콜을 통해 전송
- ex) Apache, nginx, WebtoB

### Container
- Java를 실행시킬 수 있는 서버의 SW

### WAS(Web Application Server)
- Web server + Container
- 동적인 컨텐츠(DB 조회 등)를 제공하기 위해 사용됨
- ex) Tomcat, WebLogic, Jeus
